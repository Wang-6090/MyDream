# 我做了个梦 - 梦境记录工具

一个记录梦境的小工具，记录那些醒来后即将消散的奇思妙想。

## ✨ 功能特性

- 📝 **文字记录** - 详细记录梦境内容
- 🎨 **绘画功能** - 用画笔描绘梦中场景
- 📅 **日历视图** - 按日期查看梦境分布
- 📊 **统计分析** - 梦境数据可视化
- 🔍 **智能搜索** - 快速查找梦境内容
- 🌙 **夜间模式** - 护眼暗色主题
- 💾 **数据备份** - 导出/导入 JSON 数据

## 🚀 快速开始

### 在线预览
```bash
npm start
```

### 运行测试
```bash
npm test
```

## 🛠️ 技术栈

- HTML5 / CSS3 / JavaScript (ES6+)
- localStorage 数据持久化
- Canvas API 绘图
- 响应式设计

## 📖 使用说明

1. 点击底部"+"按钮添加新梦境
2. 输入梦境描述，选择标签和情绪
3. 可选择绘制图画
4. 在列表页浏览和搜索记录
5. 在日历页按日期查看
6. 在统计页查看数据分析
