## 1. 概念与愿景

一个记录梦境的小工具，记录那些醒来后即将消散的奇思妙想。可以写字、画画、贴标签，久而久之发现自己梦境的规律——原来我经常梦见考试、原来我总梦到飞翔、原来这个梦我做过好几次了。

**核心感受**：像一本私密的梦境日记本，醒来后第一件事就是抓起来记下。

**产品价值**：
- 帮助用户记录和保存转瞬即逝的梦境
- 通过数据分析发现梦境规律和模式
- 提供创造性的表达方式（文字+绘画）
- 建立个人梦境档案，追踪睡眠质量

**目标用户**：
- 对梦境感兴趣的人群
- 希望记录梦境的用户
- 关注睡眠质量的用户

## 2. 设计语言

### 2.1 视觉风格
- **主题**：夜晚的梦幻感，像是月光洒在纸上
- **背景**：深蓝紫色渐变 (#1a1a2e → #16213e)
- **卡片**：半透明白色毛玻璃效果
- **强调色**：梦幻紫 (#9D4EDD)、星光金 (#FFD700)
- **文字**：白色/浅灰

- **标题**：24px，粗体
- **副标题**：18px，中等粗细

## 4. 功能与交互

### 4.1 记录梦境

**流程**：
1. 点击底部悬浮"+"按钮
2. 进入记录页面
3. 输入文字描述（必填）
4. 选择标签（美梦/噩梦/奇葩）
5. 选择情绪（开心/悲伤/害怕/困惑/兴奋）
6. 输入睡眠时长（可选）
7. 评分（1-5星）
8. 绘制图画（可选）
9. 点击保存

**输入字段**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| content | textarea | 是 | 梦境描述 |
| tag | select | 是 | 美梦/噩梦/奇葩 |
| mood | select | 否 | 情绪标签 |
| sleepHours | number | 否 | 睡眠时长 |
| rating | number | 否 | 1-5星评分 |
| drawing | canvas | 否 | 简笔画 |

### 4.2 简笔画功能

**工具栏**：
- 画笔工具（多种粗细）
- 颜色选择器（色环）
- 橡皮擦
- 清空画布
- 撤销/重做
- 导出图片

**画布特性**：
- 自适应尺寸
- 支持多种笔刷（钢笔、画笔、马克笔、喷笔）
- 支持绘制形状（直线、圆形、矩形）

### 4.3 梦境日历

**功能**：
- 月份切换（左右箭头）
- 有记录的日期显示彩色圆点标记
- 点击日期显示当天所有梦境
- 支持快速跳转到今天

### 4.4 关键词统计

**功能**：
- 自动分析梦境内容，提取高频词
- 展示最常出现的10个关键词
- 按标签分类统计
- 关键词云可视化

### 4.5 梦境相似检测

**算法**：
- 使用Jaccard相似度系数
- 分词后计算交集/并集比值
- 阈值设定为0.3（30%相似度）

**交互**：
- 保存时自动检测与历史梦境的相似度
- 相似度超过阈值时提示用户
- 可点击查看相似的历史梦境

### 4.6 趋势分析

**功能**：
- 显示最近7天的梦境记录趋势
- 柱状图可视化
- 鼠标悬停显示详细数据

### 4.7 数据管理

**功能**：
- 导出：生成JSON文件下载
- 导入：选择JSON文件恢复数据
- 清空：二次确认后清空所有数据

### 4.8 夜间模式

**功能**：
- 一键切换明暗主题
- 自动保存用户偏好
- 平滑过渡动画

## 6. API文档

### 6.1 本地存储API

#### 6.1.1 获取所有梦境

**方法**：`getDreams()`

**返回值**：
```javascript
{
  version: number,        // 数据版本号
  dreams: Array<Dream>,   // 梦境列表
  customTags: Array<string>, // 自定义标签
  settings: Object        // 用户设置
}
```

#### 6.1.2 添加梦境

**方法**：`addDream(dream)`

**参数**：
```javascript
{
  content: string,        // 梦境描述
  tag: string,           // 标签（good/nightmare/weird）
  mood: string,          // 情绪
  sleepHours: number,    // 睡眠时长
  rating: number,        // 评分
  drawing: string,       // 绘画数据（base64）
  createdAt: string      // 创建时间
}
```

**返回值**：`Dream`（包含自动生成的id）

#### 6.1.3 更新梦境

**方法**：`updateDream(id, updates)`

**参数**：
- `id`: string - 梦境ID
- `updates`: object - 更新的数据

**返回值**：`boolean`（更新是否成功）

#### 6.1.4 删除梦境

**方法**：`deleteDream(id)`

**参数**：
- `id`: string - 梦境ID

**返回值**：`boolean`（删除是否成功）

#### 6.1.5 导出数据

**方法**：`exportData()`

**返回值**：`string`（JSON字符串）

#### 6.1.6 导入数据

**方法**：`importData(jsonString)`

**参数**：
- `jsonString`: string - JSON格式的数据

**返回值**：`boolean`（导入是否成功）

### 6.2 相似度检测API

#### 6.2.1 检测相似梦境

**方法**：`findSimilarDreams(content, threshold = 0.3)`

**参数**：
- `content`: string - 梦境内容
- `threshold`: number - 相似度阈值（默认0.3）

**返回值**：
```javascript
[
  {
    id: string,
    content: string,
    similarity: number    // 相似度值（0-1）
  }
]
```

### 6.3 统计API

#### 6.3.1 获取统计数据

**方法**：`getStatistics()`

**返回值**：
```javascript
{
  totalCount: number,           // 总记录数
  monthCount: number,           // 本月记录数
  tagCounts: {
    good: number,               // 美梦数量
    nightmare: number,          // 噩梦数量
    weird: number               // 奇葩数量
  },
  keywords: Array<{            // 关键词统计
    word: string,
    count: number
  }>,
  trend: Array<{               // 趋势数据（最近7天）
    date: string,
    count: number
  }>
}
```

## 7. 数据库设计

### 7.1 数据模型

#### 7.1.1 Dream（梦境记录）

| 字段 | 类型 | 说明 | 约束 |
|------|------|------|------|
| id | string | 唯一标识符 | 主键，UUID格式 |
| content | string | 梦境描述 | 非空，最大10000字符 |
| tag | string | 标签 | 枚举：good/nightmare/weird |
| mood | string | 情绪 | 枚举：happy/sad/scared/confused/excited |
| sleepHours | number | 睡眠时长 | 可选，0-24 |
| rating | number | 评分 | 可选，1-5 |
| drawing | string | 绘画数据 | 可选，base64编码 |
| createdAt | string | 创建时间 | ISO8601格式 |
| date | string | 日期 | YYYY-MM-DD格式 |

#### 7.1.2 Settings（用户设置）

| 字段 | 类型 | 说明 | 默认值 |
|------|------|------|--------|
| nightMode | boolean | 夜间模式 | false |
| defaultTag | string | 默认标签 | "good" |
| remindTime | string | 提醒时间 | "21:00" |
| reminderEnabled | boolean | 启用提醒 | false |

### 7.2 存储结构

```javascript
{
  version: 2,
  dreams: [
    {
      id: "uuid-12345",
      content: "梦到自己在飞...",
      tag: "good",
      mood: "happy",
      sleepHours: 7.5,
      rating: 4,
      drawing: "data:image/png;base64,...",
      createdAt: "2024-01-15T08:30:00.000Z",
      date: "2024-01-15"
    }
  ],
  customTags: [],
  settings: {
    nightMode: false,
    defaultTag: "good",
    remindTime: "21:00",
    reminderEnabled: false
  }
}
```

### 7.3 数据迁移

**版本升级策略**：
- 检测当前数据版本
- 根据版本差异执行迁移脚本

## 8. 用户手册

### 8.1 快速开始

**安装**：
1. 下载并安装应用
2. 首次打开自动创建本地数据库
3. 开始记录你的第一个梦境

**基本操作**：

#### 添加梦境
1. 点击底部"+"按钮
2. 输入梦境描述
3. 选择标签和情绪
4. 可选：绘制图画
5. 点击保存

#### 查看梦境
- 在列表页浏览所有记录
- 使用搜索框搜索特定内容
- 点击卡片查看详情

#### 使用日历
- 左右切换月份
- 点击有标记的日期查看当天梦境

### 8.2 高级功能

#### 绘画工具
- **画笔**：选择颜色和粗细后直接绘制
- **橡皮擦**：擦除已绘制内容
- **清空**：清除整个画布
- **撤销/重做**：回退或恢复操作
- **导出**：将画作保存为图片

#### 统计分析
- **趋势图**：查看最近7天梦境频率
- **关键词云**：发现高频词汇
- **标签统计**：各类梦境比例

#### 数据管理
- **导出数据**：备份为JSON文件
- **导入数据**：从JSON文件恢复
- **清空数据**：删除所有记录（需确认）

### 8.3 常见问题

**Q: 数据保存在哪里？**
A: 数据保存在浏览器本地存储中，不会上传到云端。

**Q: 如何备份数据？**
A: 在设置页点击"导出数据"，下载JSON文件保存到安全位置。

**Q: 梦境相似检测是如何工作的？**
A: 通过分析关键词重叠程度来判断相似度，超过30%相似会提示用户。
